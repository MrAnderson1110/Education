lab_1_data_model.xml (исходники диаграмм) можно открыть с помощью сервиса:
https://www.diagrams.net/

Документ содержит несколько страниц, переключение между которыми возможно с помощью панели вкладок
в нижней части экрана приложения

Атриуты в сущности можно зацепить left click'ом и переместить в нужное место (в том числе в другую сущность)
PK — Primary Key (первичный ключ)
FK — Foreign Key (внешний ключ)
Представление отношений многие-ко-многим в базе данных показано во второй вкладке документа
